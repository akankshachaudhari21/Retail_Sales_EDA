#  Retail Sales Analysis

## Project Overview
This project focuses on analyzing a retail sales dataset using Python.
The goal is to extract meaningful insights related to revenue, customer behavior, and sales performance.

##  Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

###  Data Cleaning
- Checked for missing values  
- Converted date column to datetime format  

###  Revenue Analysis
- Total revenue generated  
- Average order value (AOV)  
- Revenue by city and region  

###  Product Analysis
- Top 10 selling products  
- Best and least performing categories  

###  Customer Behavior
- Most frequently used payment method  
- Online vs In-store sales distribution  

###  Time-Based Analysis
- Monthly revenue trend  
- Weekend vs Weekday sales comparison  

## Key Insights
- Weekend sales are slightly higher than weekday sales  
- Sales channels (Online vs In-store) are almost equally distributed  
- Certain categories contribute more to overall revenue  
- Business shows stable performance across regions  

## 📷 Visualizations
- Pie Chart (Sales Channel Distribution)  
- Line Chart (Monthly Revenue Trend)  

## 🚀 Conclusion
This project demonstrates how data analysis can help understand business performance, customer behavior, and sales trends, enabling better decision-making.
